# Heart-disease-prediction-using-QMBC-algorithms
Heart Disease Prediction using QMBC is a machine learning-based system that predicts heart disease using medical data. It combines multiple classifiers with feature selection, PCA, and a novel ensemble QMBC approach to improve accuracy and support early diagnosis.


❤️ Heart Disease Prediction Using QMBC

📌 Overview
Heart Disease Prediction Using Quine McCluskey Binary Classifier (QMBC) is a machine learning-based project that predicts the likelihood of heart disease using patient medical data. The system helps in early diagnosis and supports healthcare professionals in making better decisions.

🎯 Objective
Predict heart disease accurately using medical datasets
Reduce irrelevant features for better performance
Improve accuracy using an ensemble model (QMBC)
Assist doctors in early detection and diagnosis

🚀 Features
Uses multiple machine learning algorithms
Implements feature selection (Chi-Square, ANOVA)
Applies feature extraction using PCA
Combines predictions using QMBC method
Provides fast and reliable results
